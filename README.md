# Simon Dice

## Description
Simon Dice is a memory game inspired by the classic Simon Says game. 
The player must repeat a sequence of colors that becomes longer with each round.

This project was developed as an academic exercise to practice programming logic and event handling.

## Technologies
- Java
- JavaFX

## Features
- Random sequence generation
- Graphical interface using JavaFX
- Sequence validation
- Progressive difficulty
- Multiple difficulty levels (easy, medium, hard)
  
## How to run the project
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run the main class
